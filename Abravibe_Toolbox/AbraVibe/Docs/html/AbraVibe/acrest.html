acrest

PURPOSE

ACREST Calculate crest factor

SYNOPSIS

function C = acrest(x);

DESCRIPTION 

ACREST Calculate crest factor

       C = acrest(x);

       C           Crest factor (max(abs(x)) over std(x))

       x           Time data, if more than one column, each column is
                   computed and S is a row vector
