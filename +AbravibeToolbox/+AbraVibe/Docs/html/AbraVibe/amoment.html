amoment

PURPOSE

AMOMENT Calculate statistical moment

SYNOPSIS

function M = amoment(x,n);

DESCRIPTION 

AMOMENT    Calculate statistical moment

           M = amoment(x,n);

           M           Central moment of order n (mean of x removed)

           x           Time signal(s), in column(s)
           n           Statistical order (2 for variance, ...)
