ahann

PURPOSE

AHANN Calculate a Hanning window

SYNOPSIS

function w = ahann(N);

DESCRIPTION 

AHANN Calculate a Hanning window

           w = ahann(N);

           w           Hanning time window

           N           Blocksize, length of w

 Calculates a (correct!) Hanning window with length(N), in a column 
 vector. The ENBW is 1.5*df.

 See also aflattop winacf winenbw
