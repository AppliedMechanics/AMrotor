askewness

PURPOSE

ASKEWNESS Calculate skewness

SYNOPSIS

function S = askewness(x);

DESCRIPTION 

ASKEWNESS Calculate skewness

       S = askewness(x);

       S           Skewness (third moment over sigma^3)

       x           Time data, if more than one column, each column is
                   computed and S is a row vector
